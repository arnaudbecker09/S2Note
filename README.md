# S2Note

A small Spring Boot project designed to help learn JWT authentication and Spring Security under the hood.  
It demonstrates generating JWT tokens, protecting endpoints, and verifying requests.


## How to Start
1. Clone the repository

```bash
git clone https://github.com/arnaudbecker09/S2Note.git
cd S2Note
```

2.Run the application
```bash
./gradlew bootRun
```
